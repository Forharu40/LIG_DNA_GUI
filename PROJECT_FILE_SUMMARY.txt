4. JetsonThor.Bridge (Jetson 측 C++ 브리지 샘플)
==================================================

이 폴더는 Jetson Thor 내부에서 돌릴 수 있도록 만든 C++ 브리지 샘플이다.
GUI가 TCP로 보낸 문장을 받아 Ollama REST API에 넘기고, 결과를 다시 GUI로 돌려준다.

1) JetsonThor.Bridge\CMakeLists.txt
- C++ 브리지 프로그램 빌드 설정 파일이다.
- CMake 기반으로 jetson_ollama_bridge 실행 파일을 만든다.
- C++17 표준 사용
- main.cpp를 빌드 대상으로 등록

2) JetsonThor.Bridge\src\main.cpp
- Jetson 브리지의 실제 구현 파일이다.
- 구현 기능:
  - TCP 서버 생성 및 포트 대기
  - GUI가 보내는 JSON 라인 프로토콜 수신
  - requestId/model/prompt 파싱
  - Ollama REST 요청 생성
  - /api/generate 호출
  - 응답의 response/error 추출
  - elapsedMs 계산
  - 다시 GUI용 JSON 한 줄로 응답
- 구현 특징:
  - 외부 JSON 라이브러리 없이 단순 파서 직접 구현
  - HTTP 요청도 직접 문자열로 조립
  - Jetson에서 가볍게 샘플 구동하는 데 초점

3) JetsonThor.Bridge\README.md
- Jetson 브리지 빌드/실행 문서다.
- 포함 내용:
  - 프로토콜 예시
  - CMake 빌드 방법
  - 실행 예시
  - GUI 쪽에서 어떻게 붙여야 하는지
